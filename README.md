# Cliplog

Cliplog is a web-first clipboard dashboard. Copy text anywhere, open Cliplog, then click the import button or press `Cmd/Ctrl + V` to save it into a dated timeline.

## MVP

- Save clipboard text by click or paste shortcut
- Store clips locally in the browser
- Detect links, code, contact text, and sensitive-looking content
- Group clips by date
- Search and filter clips
- Pin or delete saved clips

## Run

```bash
npm run dev
```

Open `http://localhost:3000`.

## Deploy

The app is ready for Vercel deployment as a standard Next.js project.
